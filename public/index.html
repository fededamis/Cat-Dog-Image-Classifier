AI - Clasificador de Imágenes

La siguiente aplicación se trata de un clasificador binario de imágenes que permite distinguir imágenes de perros y de gatos, utilizando Redes Neuronales Artificiales (Deep Learning). Se trata de un algoritmo de Inteligencia Artificial. Si bien el ejemplo es bastante trivial, este mismo concepto se puede extender para la clasificación de imágenes según N etiquetas predefinidas. Por citar un caso práctico, en el siguiente paper se detalla un método de diagnóstico automatizado de COVID-19 alimentando a un algoritmo de este estilo con imágenes de rx de tórax.

Para el modelo de predicción se aplicó Transfer Learning sobre un modelo NasNet Mobile, entrenado mediante Cloud CPU en Google Colab. Se aplicaron técnicas como Data Augmentation, Early Stopping, Model Checkpoint y Learning Rate Scheduler. El valor de precisión obtenido fue de 0,9825. Se puede ver el archivo de Jupyter Notebook aquí. El backend está desarrollado en Node.js y se utilizó la librería de AI Tensorflow. El código completo se puede encontrar aquí.
